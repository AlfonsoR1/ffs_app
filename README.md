# ffs_app
This is a fantasy football application used to help track stats accurate and make accurate predictions 

## Project Setup

This project was created and tested using Python 3.11.

Required libraries are listed in `requirements.txt`.

To create and activate the virtual environment:

```powershell
py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
